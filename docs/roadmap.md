# Product Roadmap

## MVP

- Firebase Auth with Google and email/password
- Dashboard with multiple resumes
- Structured resume editor
- Drag-and-drop section ordering
- Template marketplace with search
- Local ATS scoring
- Cloud Function ATS scoring
- PDF export job placeholder
- Stripe/Paymob/Fawry checkout placeholders
- Freemium and Pro pricing surfaces

## V1

- Real auth screens and onboarding
- Firestore resume dashboard
- Tiptap rich text section editor
- Version history and restore
- Playwright PDF export worker
- Firebase Storage PDF downloads
- Premium templates
- Arabic RTL resume templates
- Admin template manager
- Paymob and Fawry production integration
- OpenAI rewrite, summary, skills, and cover-letter tools

## V2

- Job-specific resume variants
- Application tracker
- LinkedIn profile optimizer
- GitHub and Behance imports
- AI interview prep from resume
- Template analytics
- Coupon codes and student pricing
- University/career-center B2B plans

## V3

- Mobile app
- Team and white-label career center product
- Recruiter share links
- Role-specific ATS benchmarks
- Regional salary and job-market guidance
